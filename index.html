UNDER GRADUATE PROJECT

Minor project: Digital Alarm Clock with Thermometer
A digital clock was built which displayed time and date along with the ambient temperature. It was an 8051 based system which included 16X2 LCD for display, LM35 for temperature sensing and ADC0804 for interfacing sensor with 8051. It had inputs in order to adjust date and time. It was done in a group of four students.
Major project: Cell Phone Based Voting Machine with Voice Acknowledgement
The system could receive call and also attain votes from the caller in the form of DTMF tones. It would then count the received vote to the particular candidate/choice and store it in the EEPROM. It was a PIC16F877A based system which consisted of APR voice module, MT8870 DTMF decoder and an EEPROM. This Project was done by a group of four members.
